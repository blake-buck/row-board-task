row-board-task

A multi-row kanban board application

Example row

Example task dialog

Feature List

Kanban style task tracking on multiple rows Variety of options for styling tasks including icons, labels, card color, etc Can attach display photos and files to tasks Can add checklists and due dates to tasks Can sort tasks by alphabetically, date created, creation, etc

Built With

Angular v8.0.0 Node v10.15.3 with NestJS v7.0.0 MongoDB Atlas v4.2.9 Authentication provided by Amazon Cognito File storage provided by AWS S3 Hosted on AWS Lightsail Link to GitHub repository
